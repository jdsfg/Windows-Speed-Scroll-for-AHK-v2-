# Windows 极速滚轮 (Speed Scroll for AHK v2)

这是一个专为提升 Windows 浏览效率设计的 AutoHotkey v2 脚本。它通过**智能加速算法**，让你在浏览长代码、巨型日志或长文档时实现类似罗技Master 3s高端鼠标滚轮的极速顺滑体验。

告别鼠标滚轮搓到手指发酸的日子，体验真正的生产力工具。

## ⚡ 核心特性 (Features)

* **智能极速 (Smart Acceleration)**：脚本自动识别滚动意图。慢滚时保持逐行精准，快滚时瞬间触发非线性加速，一步跨越千行。
* **急停控制 (Instant Stop)**：配合阻尼算法，既有高速滑行的爽快，又能随时通过反向滚动实现“急刹车”，精准定位目标行。
* **无缝切换**：无需任何开关，完全根据你的手指滚动频率自动在“精准模式”和“极速模式”间切换。
* **特定场景优化**：在 VS Code、浏览器、PDF 阅读器中体验尤佳。
* **一键原声**：按住 `Alt` 键滚动可暂时禁用增强，恢复系统默认，方便地图缩放等操作。

## 🚀 使用方法

1.  下载本仓库的 `.ahk` 脚本。
2.  确保已安装 [AutoHotkey v2](https://www.autohotkey.com/)。
3.  双击运行，即刻生效。

## ⚙️ 极速参数调教

你可以打开脚本修改以下参数，打造最适合你手速的配置：

* `accel` (加速度)：**核心参数**。数值越大，起步越暴力，提速越快。
* `maxBurst` (速度上限)：决定了最快能滚多快。建议设为 40+ 以获得极致速度。
* `friction` (滑行距离)：控制松手后的滑行距离。
* `timeGap` (判定阈值)：决定判定为“连续快速滚动”的时间窗口。
