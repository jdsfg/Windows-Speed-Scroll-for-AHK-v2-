# Windows 惯性滚轮脚本 (Inertia Scroll for AHK v2)

这是一个轻量级的 AutoHotkey v2 脚本，旨在为 Windows 系统带来类似 macOS 或智能手机触摸屏那样的**丝滑惯性滚动**体验。

## ✨ 主要功能 (Features)

* **物理惯性模拟**：模拟真实的物理阻尼，松手后页面会自然滑行一段距离，告别生硬的停止。
* **智能加速算法**：能够感知你的滚动速度。快速连续拨动滚轮时会自动加速，浏览长文档更高效。
* **精准模式 (Precision Mode)**：当你慢速滚动时，脚本保持 1:1 的精准行数，确保阅读时不“打滑”。
* **高度可自定义**：如果不顺手，可以轻松调节摩擦力、加速度和灵敏度参数。
* **一键临时禁用**：在地图软件或 3D 建模软件中，按住 `Alt` 键滚动即可暂时禁用惯性，恢复系统原生的缩放手感。

## 🚀 使用方法 (How to Use)

1.  **环境准备**：确保你的电脑已安装 [AutoHotkey v2](https://www.autohotkey.com/) (注意必须是 v2 版本)。
2.  **下载脚本**：下载本仓库中的 `InertiaScroll.ahk` 文件。
3.  **运行**：双击脚本文件即可运行（右下角会出现绿色 H 图标）。
4.  **开机自启 (可选)**：
    * 按下 `Win + R`，输入 `shell:startup` 并回车。
    * 将脚本的快捷方式放入打开的文件夹中，即可实现开机自动加载。

## ⚙️ 参数配置 (Configuration)

用任意文本编辑器打开脚本文件，可以在头部修改以下参数以适应你的手感：

* `friction` (摩擦系数)：控制松手后滑行的距离。数值越小停得越快，数值越大滑得越远（建议范围 0.7~0.9）。
* `accel` (加速度)：控制滚动提速的快慢。数值越大，连续滚动时速度叠加越猛。
* `maxBurst` (极速上限)：限制滚轮滚动的最大速度。
* `timeGap` (判定时间)：单位毫秒。两次滚动间隔超过此时间则重置速度，用于区分“慢速阅读”和“快速浏览”。
